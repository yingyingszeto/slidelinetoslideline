  1.   3. Social Media Post Drawing (NO PURCHASE NECESSARY): To be eligible for the chance to win a PIN code via social media, participants must (i) have an active, valid and public Twitter or Instagram account, (ii) post a photo on either Instagram or Twitter of themselves wearing a 2018 New Era NFL Sideline Cap, and (iii) tag @Neweracanada and @lidscanada and hashtag #SidelinetoSideline in the social media post (collectively, each a “Valid Social Media Post”). By fully completing and submitting a Valid Social Media Post, participants will be eligible to win a PIN code from Sponsor. In Sponsor’s sole and complete discretion, and while supplies last, Sponsor will randomly select one PIN code winner daily at 11:59 p.m from all Valid Social Media Posts submitted during the Contest Period. Participants are solely responsible for Internet connectivity, software and/or hardware that may be required in order to create and/or submit any Valid Social Media Post.
